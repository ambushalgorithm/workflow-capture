# Workflow Capture

> Detects recurring workflows and prompts to create skills when reusable patterns emerge.

A meta-skill that watches for repeated patterns in your work and helps you capture them as reusable skills.

## Quick Start

```bash
# Install via ClawHub
clawhub install ambushalgorithm/workflow-capture
```

Add to `HEARTBEAT.md`:
```markdown
## Workflow Capture
- Run the workflow-capture skill to detect reusable patterns
- Check session duration (30+ minutes)
- Look for repeated tasks (2-3+ occurrences)
```

See [references/README.md](references/README.md) for full documentation.
